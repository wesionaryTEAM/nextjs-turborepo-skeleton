# Turborepo starter

## Install packages

yarn
OR npm run install

### Apps and Packages

| Packages   | Description      |
| ---------- | ---------------- |
| `admin`    | admin section    |
| `consumer` | consumer section |
| `ui`       | shared section   |

### start

| command              | Description               |
| -------------------- | ------------------------- |
| `yarn app:conusmer ` | run consumer section only |
| `yarn app:admin `    | run consumer section only |
| `ui`                 | shared section            |
